# React-Movie-Database
A React Movie Database
 
```sh
$ yarn install
```
 
```sh
$ yarn start
```
## App Features

- Search movie databse
- View movie info
- View movies by genre
- View cast
- View trailers

## Technical Features
 
- React
- Redux
- React Router
- Material UI
- Axios
- TMBD API Data
- Webpack
- ES6
